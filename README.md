# Продуктовая аналитика: Проверка статистических гипотез и анализ конверсии (CR)

## Описание проекта
Сквозной аналитический проект по исследованию продуктовых метрик и оценке потребительского спроса. В рамках работы проводился глубокий анализ воронки продаж и конверсии для различных сегментов пользователей.

## Какие задачи были решены:
* **Сегментация трафика:** Проведен сравнительный анализ конверсии (CR) для органического и платного трафика.
* **Кросс-платформенный анализ:** Выявлены различия в поведении пользователей на мобильных устройствах (Mobile) и десктопах (Desktop).
* **Географический анализ:** Исследован спрос и конверсия в разрезе ключевых регионов.
* **Проверка статистических гипотез:** Использованы математические и статистические методы для проверки значимости различий в метриках между сегментами.

## Технологический стек
* **Python** (Pandas, NumPy, SciPy)
* **Анализ данных:** Статистический анализ, проверка гипотез, сегментация данных.
